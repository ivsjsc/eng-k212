## 📱 4. Cấu trúc tương tác người dùng

### Tabs chính:

- Home  
- Learn  
- Practice  
- Resources  
- Profile

### Flow học bài:

1. Chọn module → hiển thị danh sách bài học  
2. Mỗi bài gồm: từ vựng, hội thoại, bài tập, video, tài nguyên  
3. Hoàn thành → cập nhật tiến độ → mở bài tiếp theo
